### Run docker laravel with nginx

- docker-compose up

- docker-compose exec web /bin/bash

- cat /etc/os-release

### Set permission

- chown -R www-data:www-data /var/www OR chown -R 33:33 /var/www
- chmod -R 777 /var/www

### Asset website

- Web: http://localhost

- DB: http://localhost:8888

- MailHog: http://localhost:8025
